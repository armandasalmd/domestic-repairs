#### Setup guide:

-   Open terminal and navigate to `Desktop` folder (or any other)
-   `git clone https://github.coventry.ac.uk/barkausa/5001CEM`
-   `cd 5001CEM && code .` **_To open in VS Code_**
-   `npm install`
-   `npm start` **_Done_**
